File Format Instructions

  • Sheet name must be data
  • Row 1 = headers

  • Column 2: Title
  • Column 3–7: Bullet Points (1–5)
  • Column 8: Product Description
  • Column 9: (keep blank)
  • Column 10 onward: attributes filled from selected checkboxes
